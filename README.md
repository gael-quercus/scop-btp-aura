# SCOP BTP AURA - Cartographie Interactive

Application de cartographie interactive des SCOP du BTP en Auvergne-Rhône-Alpes.

**Quercus Coopération** - Réseau des SCOP du BTP

## Fonctionnalités

- Cartographie interactive des entreprises SCOP
- Annuaire des 40 métiers du BTP
- Références de projets avec galerie photos
- Zones d'activité géographiques
- Données alimentées par Google Sheets
